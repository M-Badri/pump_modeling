# pump_modeling
Modeling of a centrifugal water pump, placed in a steam reforming plant, using OpenFoam v2206:
- 7Mln cells mesh,
- MRF zone, 
- $\Delta p = 2.1 bar$,
- $\omega = 160 rad/s$,
- $\eta = 0.6$,
- $Q = 1.097*10^{-2} m^3/s$
